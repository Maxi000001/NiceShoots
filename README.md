# 🎯 NiceShoots — Custom Crosshair Overlay

**NiceShoots** — это профессиональный, легковесный и настраиваемый прицел (оверлей) для шутеров. Программа работает поверх всех окон, обеспечивая идеальный фокус и преимущество в любой игре, от тактических шутеров до динамичных Battle Royale.

---

## 🔥 Особенности
* **Always on Top:** Надежный оверлей, который стабильно работает поверх полноэкранных оконных режимов.
* **Кастомные прицелы:** Несколько уникальных стилей прицела, подходящих под разные виды оружия и задачи.
* **Pro-настройки:** Тонкая регулировка цвета, размера, прозрачности и толщины линий прямо в приложении.
* **Оптимизация:** Приложение разработано на Electron с упором на минимальное потребление ресурсов системы (не съедает FPS).

---

## 📥 Скачивание и Установка
Для обычных пользователей (Геймеров):
1. Перейдите на страницу **[Latest Releases](https://github.com/Maxi000001/NiceShoots/releases/latest)**.
2. Скачайте файл `NiceShoot Setup 1.0.0.exe`.
3. Запустите установщик и следуйте инструкциям на экране.
4. Откройте игру в режиме "Окно без рамок" (Borderless Window) и наслаждайтесь!
